The word cloud was created with the the string of all of the motive reported by the media since 1996 after filtering out the meaningless words such as "specific","however","the"or the information that we had already known such as "motive","attack","unknown". the wordcloud package in python picks the 200 most common used non-stopwords in the given strings and assign then the frequency-related size.We use word cloud because it may help us to draw a rough outline of the motives from the criminals.
